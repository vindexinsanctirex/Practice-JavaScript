Objetivos de aprendizagem

  • Compreender o que é o DOM e como ele representa uma página HTML.
  • Selecionar elementos do DOM usando diferentes APIs (querySelector, getElementById, etc.).
  • Modificar conteúdo, atributos, classes e estilos dinamicamente.
  • Adicionar e remover elementos do DOM.
  • Lidar com eventos e callbacks (click, input, submit).

Conceitos chave

Árvore do DOM
O DOM é uma representação em árvore do documento HTML — cada elemento é um nó que pode ser lido e alterado.
Seletores
querySelector, querySelectorAll, getElementById, getElementsByClassName e getElementsByTagName.
Manipulação
textContent, innerHTML, setAttribute, classList, style, appendChild, removeChild e replaceChild.
Eventos
addEventListener, event object, propagation (bubbling/capturing) e preventDefault/stopPropagation.

Exercícios

  1. Selecione o terceiro item da lista e mude seu texto para algo criativo.
  2. Crie um contador que mostre quantos itens existem na lista e atualize-o quando a lista mudar.
  3. Implemente a edição inline de um item da lista (clique duas vezes para editar).
  4. Evite que um formulário recarregue a página ao enviar e capture os dados via JS.

Sugestão de avaliação

Pequeno projeto: criar um TODO app que permita adicionar, editar, remover e persistir itens em localStorage. Critérios: funcionamento (70%), semântica HTML (15%), estilo e UX (15%).
